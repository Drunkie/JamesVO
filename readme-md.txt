- Sophisticated gold and charcoal color palette

## 🚀 Development

### Local Development
```bash
npm start
```

### Deploy Updates
```bash
npm run deploy
```

## 📝 Content Updates

### Update Voice Samples
Replace SoundCloud URLs in `src/App.js` with your actual track embed codes.

### Update Professional Photo
Replace the hero image URL with your professional headshot.

### Customize Content
Edit testimonials, services, and contact information in `src/App.js`.
